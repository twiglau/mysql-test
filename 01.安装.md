# 安装

## 查询是否安装

```sh
rpm -qa|grep -i mysql
```

## 查看安装位置

```sh
ps -ef|grep mysql
```

## 查看 Linux 用户组

```sh
cat /etc/passwd|grep mysql
cat /etc/group|grep mysql
mysqladmin --version
```

## 启动 linux 后台服务

```sh
service mysql start
service mysql stop

p_UEV#9$%1u
```

## 设置开机自启动

```sh
chkconfig mysql on
chkconfig -list|grep mysql
```
